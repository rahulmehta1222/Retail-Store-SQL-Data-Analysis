🛍️ Retail Store Data Analysis using SQL

📊 Retail Analytics | Customer Behavior Insights | SQL Data Analysis Project 

📌 Project Overview

This project analyzes a retail store transactional dataset using SQL to uncover meaningful insights related to customer purchasing behavior, product performance, and overall sales trends.

By exploring customer demographics, transaction records, and product category information, the analysis helps identify how customers interact with the store, which product segments generate the most revenue, and how different store channels contribute to sales performance.

The project simulates real-world tasks performed by Data Analysts, where raw transactional data is transformed into actionable business insights that support better decision-making in areas such as customer targeting, product strategy, and revenue optimization.

Using SQL techniques such as joins, aggregations, filtering, and analytical queries, this project demonstrates how data can be leveraged to understand customer behavior, evaluate product demand, and improve business performance in a retail environment.

🎯 Project Objective

The goal of this project is to analyze retail data in order to:

✔ Understand customer purchasing patterns
✔ Identify high-performing product categories and subcategories
✔ Evaluate revenue contribution across store channels
✔ Analyze sales and product return behavior
✔ Extract insights that support data-driven business decisions

🗃 Dataset Description

The dataset used in this project contains three relational tables:

👤 Customers

Contains customer demographic information such as gender, date of birth, and city code.

🧾 Transactions

Includes detailed transaction records such as purchase quantity, transaction date, store type, and total transaction amount.

📦 Product Category Info

Contains product category and subcategory mappings used to classify items sold in transactions.

These tables are linked using relational keys, enabling deeper analysis of customer behavior and sales performance.

🛠 SQL Skills Demonstrated

This project showcases several important SQL techniques used by Data Analysts:

🔹 Data Exploration and Filtering
🔹 INNER JOIN for combining relational tables
🔹 Aggregation Functions (SUM, COUNT, MAX)
🔹 GROUP BY and HAVING for analytical grouping
🔹 Conditional Logic with CASE WHEN
🔹 Subqueries for dynamic analysis
🔹 Date functions for time-based insights
